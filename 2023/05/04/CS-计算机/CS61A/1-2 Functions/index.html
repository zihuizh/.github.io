CS61A Summer 2020 Lecture 2: Functions

  • Names,Assignment,and User-Defined Functions

    可以使用多个名称表示同一个函数（f&max)。
    名称可以被重新定义为其他值（max=7）。
    也就是函数本身也是一个对象，可以被赋值或者赋给其它变量。
    使用def来定义函数。

  • Environment Diagrams （Evaluation Rule for assignment suite）
    可以查看程序的具体执行情况。
    赋值的顺序是先从等号右边开始，从左到右运算；运算完成后，再进行赋值

    执行结果是什么？并不是b=3、a=3，而是b=3、a=2，也就是说并不是把第三行分成b=a+b、a=b两条语句先后执行，而是先计算a+b、b，再进行赋值。

  • Defining Functions
    定义函数所做的事：定义函数的实体并为之指定了名称。
    使用函数所做的事：在新建的函数内部环境中执行语句。（函数中的变量与主程序中无关）
    环境就是一系列的结构。

  • Q2:A Plus Abs B
    利用if、else实现输出绝对值，非常简单。
  • Q3:Two of Three
    利用min()与max()函数，可以先求$x^2+y^2+z^2$，再减去$max^2(x,y,z)$，即可。
  • Q4:Largest Factor
    求一个正整数的最大因数，从1开始到x-1，进行遍历，如果能整除，则更新因数factor。
  • Q5:If Function vs Statements
    这里主要要提醒的是函数作为参数传入时是会被执行的，因此if_function和if实际上不同，if_function会将condition, true_result, false_result都执行一遍。 ^3dc5e5
  • Q6:Hailstone
    取正整数x，
    $$\left{\begin{align}x &= x/2,\ x \ is \ even.\
    x &= 3*x+1, \ x \ is \ odd.\end{align}\right.$$
    使用if(x%2==0)进行判断，之后操作即可。
