Return

  • search函数
    返回主程序中并提供函数执行的值，函数执行至return为止。 还可以再简化： search()函数的功能是从0开始遍历正整数，寻找到符合f(x)的最小整数并输出。[[00-CS61A#Higher-Order Functions]]，这是一个高阶函数。
  • 高阶函数 inverse是这样一个函数：
    它输出的是一个函数：lambda y: search(lambda x: f(x) == y)
    输出的这一函数可以根据输入值y寻找符合f(x)==y的最小正整数x。
    也就是这一函数的功能是给定y值，调用search()函数，遍历找到使f(x)==y的x值。
    这也事实上就是f(x)的反函数（当然只能返回最小x值，而不是所有x值）。
    因此便有sqrt = inverse(square)
    此中妙处，需要细细体会。

Control (Evaluation Rule for Call Expression)

是否可以构造这样一个函数？ [[#Homework 1 Variables Functions Control]]Q5
if(header expression , "if" suite , "else" suite)
即三个参数分别为要判断的表达式、为真时执行的语句、为假时执行的语句。

  • 这是不行的。因为if函数在调用时会自动执行各项参数，因此三个表达式(语句)都会被执行一遍，也就不能完成函数的功能。
  • 调用时的执行顺序为首先执行各项参数语句，再将执行结果带入到函数当中。 上面所示即为if()函数的具体结构，那么在下面所示的函数中来应用它： 这样是可以运行成功的，即正数开方就是正常运算，负数开方则实部为0。但是如果采用if_(x > 0, sqrt(x), 0.0) 函数，计算real_sqrt(-4)则会出现报错，为什么？
    就是因为执行时会顺序首先是计算-4>0，sqrt(-4)以及0.0这三个参数，再去执行函数，尽管函数体中sqrt(-4)不会被执行，但是参数已经预先被执行了，计算过程中sqrt(-4)会出现报错。而如果使用正常的if语句，则return sqrt(-4)根本不会执行，这就是两个的区别。
    重点就是掌握call expression的执行顺序。

Control Expressions

  • logical operators

    • or，返回遇到的第一个真值，如果没遇到真值，就返回遇到的最后一个假值。
    • not，返回True / False。
